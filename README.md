# Customer Churn Prediction

## Introduction

This project demonstrates how basic Python programming concepts can be applied to solve a real-world business problem. It analyzes customer information to identify customers who are at risk of churn and determines whether they should receive a retention offer.


## Project Goal

The goal of this project is to use Python fundamentals to:

- Store customer information using variables.
- Calculate annual subscription revenue.
- Analyze customer behavior.
- Predict customer churn risk.
- Determine retention offer eligibility.


## Dataset Description

The project uses two sample customer records containing:

- Customer Name
- Monthly Subscription Fee
- Months with Company
- Customer Care Calls
- Missed Payments
- Monthly Data Usage
- Minimum Expected Data Usage
- Premium Subscriber Status


## Tools Used

- Jupyter Notebook


## Methods

The project was completed using:

- Variables
- Arithmetic Operators
- Assignment Operators
- Comparison Operators
- Logical Operators
- Print Statements

No `if` statements, loops, functions, lists, or dictionaries were used.
